## Using vue CLI

  sudo npm add -g @vue/cli  ->  to install vue cli
  sudo npm i -g @vue/cli    ->  to update vue cli
  
  vue create <vue-app-name>
  
  npm run serve  ->  for development server
  
  # For production build
  npm run build   ->  creates a dist folder
  sudo npm i -g serve  ->  prod deploy code test,  serve dist folder
  serve -s dist  -> prod serve
  
  
  



## v-if  vs v-show
-- Use v-if if the condition won't change that often. It will not load/render the element until the condition is met and renders element as condition is met 
-- Use v-show if you want to toggle the condition more often. v-show does not remove the element from the DOM if your condition is false
